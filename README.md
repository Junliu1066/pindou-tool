# 拼豆图纸生成器

把图片转成拼豆坐标图的纯静态网页工具。上传图片 → 自动匹配色号 → 生成可交互坐标图。所有计算在用户浏览器本地完成，**图片不上传服务器**。

## 文件

| 文件 | 作用 |
|---|---|
| `index.html` | 主工具（整个应用就一个文件） |
| `admin.html` | 后台管理面板（看本机使用/错误日志） |

## 功能

- 图片转坐标图，自动匹配 MARD 风格色号
- 点击色号高亮，其余颜色仍可见
- 行/列坐标定位、选中色高亮所在行列
- 缩放：按钮 / 长按连续 / 双指捏合 / 触控板捏合，悬浮控件可拖动
- 辅助线：每 5 格（蓝）、每 10 格（红），对应拼豆板大小格
- 成品预览 / 坐标图 两种视图，导出 PNG、分色图层
- 移动端 / 平板适配
- 前端错误日志，出错时用户可一键复制回报

## 本地预览

```bash
python3 -m http.server 8080
# 浏览器打开 http://localhost:8080
```

## 部署（自己的服务器，Nginx）

```bash
# 把 index.html / admin.html 放到网站根目录，例如 /var/www/pindou/
# Nginx 配置 root 指向该目录即可，纯静态、零后端。
```

## 改密码

后台默认密码在 `admin.html` 顶部 `const ADMIN_PW='admin888'`，部署前改掉。
（前端密码仅防误入，非强安全；如需真正安全的后台请加后端登录。）

## 版本

当前版本号写在 `index.html` 的 `window.APP_VERSION`。改动后记得同步更新，方便日志对应。
