🖥️ 服务器日志（看真实访问量 / 改密码 / 修 bug）

部署到自己服务器后，跨用户的真实数据都在服务器访问日志里，无需第三方统计：

# 总访问次数
grep index.html /var/log/nginx/access.log | wc -l

# 独立访客（按 IP 去重）
awk '{print $1}' /var/log/nginx/access.log | sort -u | wc -l

# 可视化报表（装一次 GoAccess，免费、本地）
sudo apt install -y goaccess
goaccess /var/log/nginx/access.log -o /var/www/pindou/report.html --log-format=COMBINED
